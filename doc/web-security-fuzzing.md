# Web Security & Fuzzing

- [XSStrike](https://github.com/s0md3v/XSStrike) - XSStrike is the most advanced XSS scanner, capable of detecting and exploiting Cross-Site Scripting vulnerabilities.
- [Corsy](https://github.com/s0md3v/Corsy) - Corsy is a CORS misconfiguration scanner that helps identify improper CORS implementations.
- [Photon](https://github.com/s0md3v/Photon) - Photon is an incredibly fast crawler designed for Open Source Intelligence (OSINT) gathering from web targets.
- [Gobuster](https://github.com/OJ/gobuster) - Gobuster is a tool used to brute-force URIs (directories and files), DNS subdomains, and virtual host names.
- [FuzzDB](https://github.com/fuzzdb-project/fuzzdb) - FuzzDB is a dictionary of attack patterns and primitives for black-box application fault injection and fuzzing.
- [PS-Fuzz](https://github.com/prompt-security/ps-fuzz) - PS-Fuzz is a fuzzing tool specifically designed for testing prompt injection attacks in AI systems.

# Active Directory & Lateral Movement

- [CrackMapExec](https://github.com/byt3bl33d3r/CrackMapExec) - CrackMapExec is a swiss army knife for pentesting networks, with extensive Active Directory support.
- [Impacket](https://github.com/fortra/impacket) - Impacket is a collection of Python classes for working with network protocols, including SMB and Kerberos.
- [NoPac](https://github.com/Ridter/noPac) - NoPac is a tool for exploiting CVE-2021-42278 and CVE-2021-42287 in Active Directory environments.
- [CVE-2020-1472](https://github.com/dirkjanm/CVE-2020-1472) - A tool for exploiting the Zerologon vulnerability (CVE-2020-1472) in Active Directory.

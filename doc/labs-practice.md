# Labs, Practice & Vulnerable Apps

- [Metasploitable3](https://github.com/rapid7/metasploitable3) - Metasploitable3 is a vulnerable machine image designed for testing Metasploit and other security tools.
- [Security Shepherd](https://github.com/OWASP/SecurityShepherd) - Security Shepherd is a web application security training platform with various challenges.
- [Vulhub](https://github.com/vulhub/vulhub) - Vulhub provides pre-built vulnerable environments based on Docker for practicing security techniques.
- [Bugcrowd University](https://github.com/bugcrowd/bugcrowd_university) - Bugcrowd University offers training resources and challenges for bug bounty hunters.

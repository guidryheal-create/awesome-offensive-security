# Network Scanning & Enumeration

- [Nmap](https://github.com/nmap/nmap) - Nmap is the Network Mapper, a free and open-source utility for network discovery and security auditing.
- [RustScan](https://github.com/RustScan/RustScan) - RustScan is a modern, fast port scanner written in Rust, designed to be the fastest port scanner available.
- [Masscan](https://github.com/robertdavidgraham/masscan) - Masscan is a TCP port scanner that can scan the entire Internet in under 6 minutes, transmitting 10 million packets per second.
- [ZMap](https://github.com/zmap/zmap) - ZMap is a fast single packet network scanner designed for Internet-wide network surveys.

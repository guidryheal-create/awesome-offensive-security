# Red Team Toolkits & Automation

- [AutoSploit](https://github.com/NullArray/AutoSploit) - AutoSploit is an automated exploitation framework that leverages Metasploit for mass exploitation.
- [Sn1per](https://github.com/1N3/Sn1per) - Sn1per is an automated pentest framework for reconnaissance and vulnerability scanning.
- [AutoRecon](https://github.com/Tib3rius/AutoRecon) - AutoRecon is a multi-threaded network reconnaissance tool that performs automated enumeration.
- [Osmedeus](https://github.com/j3ssie/Osmedeus) - Osmedeus is a fully automated offensive security framework for reconnaissance and vulnerability scanning.
- [Red Teaming Toolkit](https://github.com/infosecn1nja/Red-Teaming-Toolkit) - A collection of tools and resources for red teaming operations.

# Threat Hunting & Detection

- [Osquery](https://github.com/osquery/osquery) - Osquery is an operating system instrumentation framework for Windows, macOS, and Linux, using SQL queries.
- [Zeek](https://github.com/zeek/zeek) - Zeek is a powerful network security monitor that inspects network traffic for signs of suspicious activity.
- [Suricata](https://github.com/OISF/suricata) - Suricata is a high-performance Network IDS, IPS and NSM engine, capable of real-time intrusion detection.
- [CrowdSec](https://github.com/crowdsecurity/crowdsec) - CrowdSec is a collaborative security engine that blocks malicious IPs using crowd-sourced intelligence.
- [Maltrail](https://github.com/stamparm/maltrail) - Maltrail is a malicious traffic detection system that uses various sources to identify threats.

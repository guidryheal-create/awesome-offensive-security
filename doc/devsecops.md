# DevSecOps & Code Security

- [CodeQL](https://github.com/github/codeql) - CodeQL is a semantic code analysis engine that helps find security vulnerabilities and errors in code.
- [Gitleaks](https://github.com/gitleaks/gitleaks) - Gitleaks is a tool for detecting and preventing hardcoded secrets like passwords and API keys in Git repositories.
- [Shhgit](https://github.com/eth0izzle/shhgit) - Shhgit is a tool that finds secrets and sensitive files across GitHub using regex and entropy.
- [Hadolint](https://github.com/hadolint/hadolint) - Hadolint is a Dockerfile linter that helps improve Docker images by checking for best practices and common mistakes.

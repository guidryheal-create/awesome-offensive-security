# Reconnaissance & OSINT

- [SpiderFoot](https://github.com/smicallef/spiderfoot) - SpiderFoot is the most complete OSINT collection and reconnaissance tool, automating the gathering of intelligence from various sources.
- [Amass](https://github.com/OWASP/Amass) - Amass performs in-depth attack surface mapping and asset discovery using open-source intelligence gathering techniques.
- [theHarvester](https://github.com/laramies/theHarvester) - theHarvester is a tool for gathering emails, subdomains, and names from public sources.
- [Sherlock](https://github.com/sherlock-project/sherlock) - Sherlock is designed to hunt down social media accounts by username across multiple social networks.
- [Awesome OSINT](https://github.com/jivoi/awesome-osint) - A curated list of amazingly awesome Open Source Intelligence (OSINT) tools and resources.
- [Censys Python](https://github.com/censys/censys-python) - A Python library for interacting with the Censys search engine to discover internet-connected devices.
- [MCP Shodan](https://github.com/BurtTheCoder/mcp-shodan) - An integration tool for Shodan within the MCP framework for internet-wide scanning.
- [MCP ZoomEye](https://github.com/zoomeye-ai/mcp_zoomeye) - ZoomEye integration for MCP, providing access to global device and service discovery.

# Rootkits & Kernel-Level

- [Diamorphine](https://github.com/m0nad/Diamorphine) - Diamorphine is an LKM (Loadable Kernel Module) rootkit for Linux that provides advanced hiding capabilities.
- [Reptile](https://github.com/f0rb1dd3n/Reptile) - Reptile is an LKM rootkit for Linux, offering rootkit functionality for stealth operations.
- [Azazel](https://github.com/chokepoint/azazel) - Azazel is a userland rootkit that provides backdoor functionality and process hiding.
- [R77 Rootkit](https://github.com/bytecode77/r77-rootkit) - R77 is a fileless ring 3 rootkit that operates in user mode for Windows systems.
- [Awesome Windows Rootkits](https://github.com/0xd3xt3r/awesome-windows-rootkits) - A curated list of Windows rootkits and related research.
- [Awesome Linux Rootkits](https://github.com/milabs/awesome-linux-rootkits) - A collection of Linux rootkits and kernel-level malware.

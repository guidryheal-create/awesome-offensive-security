# Credential Attacks & Cracking

- [Hate_Crack](https://github.com/trustedsec/hate_crack) - Hate_Crack is a Hashcat cracking tool designed for distributed password cracking operations.
- [Gocrack](https://github.com/fireeye/gocrack) - Gocrack is a distributed password cracking tool that leverages cloud computing resources.
- [Bruteforce Wallet](https://github.com/glv2/bruteforce-wallet) - A tool for brute-forcing cryptocurrency wallets using various attack methods.
- [JWT Cracker](https://github.com/lmammino/jwt-cracker) - JWT Cracker is a simple tool for brute-forcing JWT (JSON Web Token) secrets.

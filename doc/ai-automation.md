# AI & Automation in Security

- [PentestGPT](https://github.com/GreyDGL/PentestGPT) - PentestGPT is a GPT-powered penetration testing tool that assists in automated security assessments.
- [Pentest AI Agents](https://github.com/0xSteph/pentest-ai-agents) - A collection of AI agents designed to enhance penetration testing workflows.
- [Pentagi](https://github.com/vxcontrol/pentagi) - Pentagi is an AI-powered penetration testing tool that automates various security testing tasks.
- [CAI](https://github.com/aliasrobotics/cai) - CAI (Cybersecurity AI) is an AI-powered tool for cybersecurity analysis and automation.

# MITM & Traffic Analysis

- [Mitmproxy](https://github.com/mitmproxy/mitmproxy) - Mitmproxy is an interactive HTTPS proxy for intercepting and modifying HTTP/HTTPS traffic.
- [Stenographer](https://github.com/google/stenographer) - Stenographer is a packet capture solution designed for high-performance network traffic logging.
- [Ngrep](https://github.com/jpr5/ngrep) - Ngrep is a network grep tool that allows searching for patterns in network traffic.

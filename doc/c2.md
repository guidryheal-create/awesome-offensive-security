# Command & Control (C2)

- [TrevorC2](https://github.com/trustedsec/trevorc2) - TrevorC2 is a client/server command and control framework designed for red team operations.
- [Dnscat2](https://github.com/iagox86/dnscat2) - Dnscat2 is a DNS tunnel tool that can be used for command and control over DNS.
- [ExternalC2](https://github.com/ryhanson/ExternalC2) - ExternalC2 is a framework for integrating external C2 channels into existing tools.
- [External C2 Framework](https://github.com/Und3rf10w/external_c2_framework) - A framework for building external command and control systems.
